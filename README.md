# Node JS - Digital Ocean Spaces File Upload

> Nodejs file upload to Digital Ocean Spaces with Multer and Aws Sdk

### Technologies Used

1. Node JS
2. Express JS
3. AWS S3

### Steps to Run the App

- Add the .env file

- `touch .env`

- Add environment variables in the .env file

- Finally Start the App

1. `npm install`

2. `npm start`
